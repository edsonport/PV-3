## Instalação e Uso

### Requisitos

- Navegador web moderno (Chrome, Firefox, Safari, Edge)
- Servidor web (para desenvolvimento local)

### Desenvolvimento Local

1. Clone o repositório:
```bash
git clone https://github.com/seu-usuario/edson-porto-trabalhista.git
cd edson-porto-trabalhista
```

2. Inicie um servidor local:
```bash
# Python 3
python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000

# Node.js (com http-server)
npx http-server
```

3. Acesse em seu navegador:
```
http://localhost:8000
```

### Publicação

1. Faça push para seu repositório GitHub
2. Configure GitHub Pages nas configurações do repositório
3. A página será publicada automaticamente em `https://seu-usuario.github.io/edson-porto-trabalhista/`

## Customização

### Alterar Número de WhatsApp

Edite `script.js`:
```javascript
const WHATSAPP_NUMBER = 'seu_numero_aqui'; // Formato: 5551993615313
```

### Alterar Mensagem do WhatsApp

Edite `script.js`:
```javascript
const WHATSAPP_MESSAGE = 'Sua mensagem aqui';
```

### Alterar Cores

Edite `styles.css`:
```css
:root {
    --navy-blue: #1a2332;
    --gold: #c9a961;
    /* ... outras cores ... */
}
```

### Alterar Tipografia

Edite o `<head>` do `index.html`:
```html
<link href="https://fonts.googleapis.com/css2?family=SuaFonte:wght@400;600&display=swap" rel="stylesheet">
```
